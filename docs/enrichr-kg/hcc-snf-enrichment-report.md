# Functional Enrichment Analysis with Enrichr-KG

This section contains the biological networks generated via **Enrichr-KG** for the DEGs (UP and DOWN regulated) of each of the 6 clusters identified through SNF and Spectral Clustering.

Each condition features two layouts:
1. **Force-directed (`_graph`)**: Optimizes functional module identification and connection density.
2. **Geometric (`_geometric`)**: Provides a regular spatial distribution to reduce label overlapping.
